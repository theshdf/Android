#### 接口和抽象类的区别 
> 1. 接口中的方法都是抽象的，都没有方法体，抽象类中的方法可以是抽象的，可以有方法体。在架构中接口用来定义没有实现的方法，抽象类用来封装通用的代码，实现未实现的方法，同时暴露出新的抽象方法
